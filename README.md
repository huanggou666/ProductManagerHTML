# 商品管理工具

一个简洁实用的本地商品管理表格工具，基于 HTML 开发，可直接在浏览器中运行。

## 功能特性

### 📦 商品管理
- 添加商品（标题、标签、商品地址、推广页地址、所属店铺、备注、图片）
- 编辑商品 - 直接在表格内完成，无需跳转
- 删除商品 - 带确认提示
- 搜索商品 - 支持按标题、标签、店铺、备注搜索

### 📋 表格功能
- 点击任意单元格快速复制内容到剪贴板
- 商品图片 70px 宽度展示，等比例缩放
- 商品标题和常用视频Tag 列宽较大，方便查看长文本
- 悬停显示复制提示图标

### 💾 数据管理
- 本地存储 - 数据保存在浏览器 localStorage，刷新不丢失
- 导出数据 - 支持导出为 JSON 文件备份
- 导入数据 - 支持从 JSON 文件恢复数据

### 📊 统计
- 实时显示商品总数

## 表格字段

| 字段 | 说明 | 宽度 |
|------|------|------|
| 商品图片 | 70px 宽度展示 | 固定 |
| 商品标题 | 商品名称 | 较宽 |
| 常用视频Tag | 标签，多个用逗号分隔 | 较宽 |
| 商品地址 | 商品链接 | 标准 |
| 推广页地址 | 推广页面链接 | 标准 |
| 所属店铺 | 店铺名称 | 标准 |
| 备注 | 备注信息 | 标准 |
| 操作 | 编辑/删除按钮 | 固定 |

## 使用方法

1. 直接用浏览器打开 `index.html` 文件即可使用
2. 无需安装任何依赖或服务器

## 数据存储

数据存储在浏览器本地 localStorage 中：
- 不同浏览器数据独立
- 清除浏览器缓存会丢失数据
- 建议定期使用导出功能备份数据

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
- localStorage API

## 界面预览

- 简洁的卡片式设计
- 响应式布局适配不同屏幕
- 悬停效果提升操作体验
- Toast 提示反馈操作结果
